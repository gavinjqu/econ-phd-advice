# Advice for PhD Students in Economics

A curated, community-maintained collection of resources and advice for economists — from undergraduates exploring the PhD path to students on the job market.

**Live site:** [https://gjqu.github.io/econ-phd-advice/](https://gjqu.github.io/econ-phd-advice/)

## Origins

This project was originally a [Google Sites page](https://sites.google.com/view/econgradadvice/) created by [Chris Roth](https://www.chris-roth.com/) and [David Schindler](https://www.davidschindler.org/). It has been migrated to a Quarto website and expanded to include resources for undergraduates considering a PhD.

## Contributing

Contributions are welcome! You can:

- **Add a resource** — edit the relevant `sections/*.qmd` file and submit a PR
- **Report a dead link** — open an issue with the section name and broken URL
- **Suggest a new section** — open an issue for discussion before creating a PR
- **Fix formatting / typos** — PRs welcome, no issue needed

## Local development

```bash
# Preview locally
quarto preview

# Build the site
quarto render
```

Requires [Quarto](https://quarto.org/) >= 1.4.
